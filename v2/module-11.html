Scenario-Based Assessment Examples

Scenario 1 — Customer Information

Scenario

A support employee wants to use AI to summarise a customer complaint email. The email contains:

  • names
  • account references
  • sensitive customer information.

The employee plans to use a free public AI chatbot.

Question

What should the employee do?

Correct Answer

B

  1. Use the tool because it improves efficiency.
  2. Remove sensitive information and use an approved enterprise AI platform if organisational policy allows.
  3. Upload the email without changes because the AI provider is well known.
  4. Share the email from a personal account instead.

Scenario 2 — AI-Generated Report

Scenario

A manager uses AI to generate a business summary for leadership. The AI output:

  • looks professional
  • includes statistics
  • provides operational recommendations.

The manager has not verified the information.

Question

What should happen next?

Correct Answer

B

  1. Send the report immediately because the AI sounds confident.
  2. Verify important information, review the recommendations and apply human oversight.
  3. Assume the AI checked the information automatically.
  4. Remove any mention that AI was used.

Scenario 3 — Suspicious Request

Scenario

An employee receives an urgent Teams message appearing to come from a senior executive requesting immediate payment approval. The message:

  • sounds realistic
  • references genuine projects
  • pressures fast action.

Question

What is the safest response?

Correct Answer

B

  1. Approve immediately due to urgency.
  2. Verify using another approved communication channel and follow organisational process.
  3. Ignore approval procedures because the request appears genuine.
  4. Forward banking details without verification.

Scenario 4 — AI Plugin Discovery

Scenario

An employee discovers a browser extension automatically sending webpage content to an external AI service. No governance approval exists.

Question

What should the employee do?

Correct Answer

C

  1. Ignore it because no issue has happened yet.
  2. Continue using it because it saves time.
  3. Escalate the issue according to organisational governance or security processes.
  4. Recommend everyone else installs it.

Scenario 5 — Recruitment Decision

Scenario

A hiring manager uses AI to score job candidates automatically. The manager notices the AI consistently ranks certain candidates lower without clear reasoning.

Question

What is the safest approach?

Correct Answer

B

  1. Trust the AI because it improves efficiency.
  2. Review the process critically, apply human oversight and follow organisational recruitment governance.
  3. Allow AI to make the final hiring decision independently.
  4. Ignore fairness concerns because the tool is automated.

  4. "Unapproved AI usage is acceptable if productivity improves."

Operational judgement

  1. Using approved enterprise AI within policy.
  2. Following governance review processes.
  3. Using unapproved AI tools or personal AI accounts for business activity.
  4. Completing AI awareness training.
