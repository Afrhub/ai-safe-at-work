## Cadence

- One sector overlay per quarter, on top of the role-track cadence.
- Each overlay maps onto the existing universal course (modules 1–11) — it does **not** replace it. The overlay adds sector-specific never-paste rules, sector-specific incident protocols, sector-specific regulator routes.
- Each overlay is its own Governance Pack SKU (Pillar 2 in the commercial framework).
