Using AI? Pause first.

AI tools are useful, and they remember, get things wrong, and leak. Run these four questions every time.

  1. Would I be happy for this to appear on the front page?
  2. Is there personal, customer or confidential data in what I'm pasting?
  3. Am I treating the output as a draft I will check, not as fact?
  4. Is this an approved tool for this kind of work?

✕ Never paste

  • Customer or patient records
  • Passwords, keys or contracts
  • Anything marked confidential
  • Special-category personal data

✓ When unsure

  • Stop before you paste
  • Ask your manager or the DPO
  • Use an approved tool only
  • Report a slip, fast beats hidden
